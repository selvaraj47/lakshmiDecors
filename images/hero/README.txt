HERO IMAGES FOLDER
==================

Please add your hero background images to this folder with the following names:
- hero1.jpg
- hero2.jpg
- hero3.jpg
- hero4.jpg
- hero5.jpg

The images will automatically rotate every 5 seconds in the hero section.

Recommended image specifications:
- Format: JPG or PNG
- Size: 1920x1080 pixels or larger (will be automatically resized)
- Aspect ratio: 16:9 or similar
- File size: Optimized for web (under 500KB each)

If images are not found, the hero section will use a gradient background as fallback.
